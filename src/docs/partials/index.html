介绍

本系统包括以下模块，点击连接查看使用详情。

  • 1、基本状况查看
  • 2、页面详情模块
  • 3、js错误日志模块
  • 4、接口错误日志模块
  • 5、安卓插件化模块
  • 6、消息推送模块

’这厮‘能做什么？

      收集客户设备详细信息，对’安卓插件，消息推送‘模块进行管理，针对’js错误信息，接口信息、页面详情信息‘进行热备份，每天日志导出， 让前端开发尽快定位问题。

基本状况查看

包含《页面js异常、接口异常、下载量》图标统计。

页面详情模块

页面报错详细信息展示。

js错误日志模块

详细js错误信息展示。

接口错误日志模块

详细接口错误信息展示《后期对接前端mock系统》。

安卓插件化模块

  • 1、根据首页提示，进入相应的页面。”登录 == 注册 == docs == 无密码进入’。
  • 2、if ( 密码 ) {
          进入；
    } else {
          注册；
    }
  • 3、选择 “Plug-in For Android‘ 模块，进行’添加项目‘功能。
  • 4、选择’增加插件‘功能，增加插件组。
  • 5、点击增加的插件’编辑‘按钮，增加详情插件。

消息推送模块

使用说明同安卓插件化管理 3、4、5 步。
